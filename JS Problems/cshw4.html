Graph Data Structure

A graph data structure is a nonlinear data structure consisting of nodes and vertices. It contains a finite set of nodes and vertices whiche are connected to each other.
The common terminologies used in graph are root, nodes, vertices, edge, parent node, child node, tree, leaf etc. Types of graphs there are many types some of them are as follows:

  • Undirected Graph: Its a graph where edges are bidirectional meaning the edges have no specific direction.
  • Directed Graph: The edges are uni-directional or point to one specific direction
  • Tree: Is an undirected graph
  • Weighte Graph: Each edge has beena assigned a value

    Psuedo Code for BFS
  • Initialize a queue
  • Add the first node to queue
  • Dequeue it to the visited array and add the adjacent nodes to the queue
  • Dequeue the node from front of queue add to visited and add a new adjacent node to the tail of the queue
  • Repeat Step 3 and 4 until the queue is empty
    • This way the child nodes are visited first then the traversal moves onto the next layer child nodes.

    Psuedo Code for DFS
  • Initialize a Stack
  • Add first node to the stack
  • Pop the top element and add that to visited array
  • Push the adjacent elements to the stack on top of each other
  • Pop the top adjacent node element and push to visited array and push a new adjacent element on top of stack
  • Repeat the last two steps until queue is empty
    • This way the graph traversal happens in one direction and when completed the traversal on one side it moves to the top layer and then continues on that tree towards leaf.
